# Asset Pricing Project 02: Fama-French 3-Factor Model Analysis

## ðŸ“Œ Project Overview
This project performs an empirical test of the **Fama-French Three-Factor Model** on **Apple Inc. (AAPL)**. 
While the standard CAPM relies solely on Market Beta, this project introduces **Size (SMB)** and **Value (HML)** factors to attribute the portfolio's returns to specific risk exposures.

**Research Question:** Is Apple's performance driven by market risk, size premiums, or value premiums?

## ðŸ”¬ Methodology
* **Asset:** Apple Inc. (AAPL)
* **Benchmark:** Fama-French Research Factors (Mkt-RF, SMB, HML)
* **Timeframe:** 2020-01-01 to 2024-01-01
* **Frequency:** Monthly Excess Returns
* **Model:** Ordinary Least Squares (OLS) Regression
    $$E(R_i) - R_f = \alpha + \beta_{mkt}(E(R_m) - R_f) + \beta_{s}SMB + \beta_{v}HML$$
